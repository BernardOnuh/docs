## What is StableF Bridge?

StableF Bridge is a powerful API that enables seamless cross-chain USDT transfers using smart wallets and LayerZero protocol. Bridge USDT between Ethereum Sepolia and Base Sepolia testnets with automatic execution.

## How It Works

<Steps>
  <Step title="Create Bridge Request">
    Make a POST request to `/bridge/create` with your source chain, destination chain, and amount
  </Step>
  <Step title="Get Deposit Address">
    Receive a unique smart wallet address to deposit your USDT
  </Step>
  <Step title="Send USDT">
    Transfer USDT to the provided deposit address on the source chain
  </Step>
  <Step title="Automatic Bridge">
    The system detects your deposit and automatically bridges to the destination
  </Step>
</Steps>

## Supported Networks

- **Ethereum Sepolia** (Testnet)
- **Base Sepolia** (Testnet)
