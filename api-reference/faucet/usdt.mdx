---
title: "Request USDT Faucet"
api: "POST /faucet/usdt"
description: "Get testnet USDT for testing the bridge"
---

## Request Body

<ParamField body="address" type="string" required>
  Ethereum address to receive testnet USDT
</ParamField>

<ParamField body="chain" type="string" required>
  Network where USDT should be sent. Must be either `sepolia` or `base-sepolia`
</ParamField>

<ParamField body="amount" type="string">
  Amount of USDT to request. Defaults to "10" if not specified
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the faucet request was successful
</ResponseField>

<ResponseField name="txHash" type="string">
  Transaction hash of the USDT transfer
</ResponseField>

<ResponseField name="recipient" type="string">
  Address that received the USDT
</ResponseField>

<ResponseField name="chain" type="string">
  Network where USDT was sent
</ResponseField>

<ResponseField name="amount" type="string">
  Amount of USDT sent
</ResponseField>

<ResponseField name="usdtToken" type="string">
  Address of the USDT token contract
</ResponseField>

<ResponseField name="explorerUrl" type="string">
  Block explorer link to view the transaction
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="blockNumber" type="string">
  Block number where transaction was included
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://your-api.com/faucet/usdt \
  -H "Content-Type: application/json" \
  -d '{
    "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
    "chain": "base-sepolia",
    "amount": "10"
  }'
```

```javascript JavaScript
const response = await fetch('https://your-api.com/faucet/usdt', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    address: '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb',
    chain: 'base-sepolia',
    amount: '10'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    'https://your-api.com/faucet/usdt',
    json={
        'address': '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb',
        'chain': 'base-sepolia',
        'amount': '10'
    }
)

data = response.json()
print(data)
```

## Using with Bridge

After receiving USDT from the faucet, you can use it to test the bridge:

<Steps>
  <Step title="Request USDT">
    Use this endpoint to get testnet USDT
  </Step>
  <Step title="Wait for Confirmation">
    Wait for the transaction to be confirmed (usually 10-30 seconds)
  </Step>
  <Step title="Create Bridge">
    Use the [Create Bridge](/api-reference/bridge/create) endpoint to start bridging
  </Step>
</Steps>
