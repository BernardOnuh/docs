---
title: "Create Bridge Request"
api: "POST /bridge/create"
description: "Create a new cross-chain USDT bridge request"
---

## Request Body

<ParamField body="sourceChain" type="string" required>
  Source blockchain network. Must be either `sepolia` or `base-sepolia`
</ParamField>

<ParamField body="destinationChain" type="string" required>
  Destination blockchain network. Must be either `sepolia` or `base-sepolia` (different from source)
</ParamField>

<ParamField body="amountUSDT" type="string" required>
  Amount of USDT to bridge (in USDT, not wei). Example: "10" for 10 USDT
</ParamField>

<ParamField body="destinationWallet" type="string" required>
  Ethereum address where USDT will be sent on the destination chain
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the bridge request was created successfully
</ResponseField>

<ResponseField name="sessionId" type="string">
  Unique identifier for this bridge session. Use this to check status
</ResponseField>

<ResponseField name="depositAddress" type="string">
  Smart wallet address where you should send USDT on the source chain
</ResponseField>

<ResponseField name="smartWalletAddress" type="string">
  The smart wallet contract address (same as depositAddress)
</ResponseField>

<ResponseField name="sourceChain" type="string">
  Confirmed source chain
</ResponseField>

<ResponseField name="destinationChain" type="string">
  Confirmed destination chain
</ResponseField>

<ResponseField name="amountUSDT" type="string">
  Confirmed amount to bridge
</ResponseField>

<ResponseField name="destinationWallet" type="string">
  Confirmed destination wallet address
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable instruction message
</ResponseField>

<ResponseField name="instructions" type="object">
  Step-by-step instructions for completing the bridge
  
  <Expandable title="properties">
    <ResponseField name="step1" type="string">
      Send USDT instruction
    </ResponseField>
    <ResponseField name="step2" type="string">
      Automatic execution notice
    </ResponseField>
    <ResponseField name="step3" type="string">
      Destination delivery notice
    </ResponseField>
  </Expandable>
</ResponseField>

## Next Steps

After creating a bridge request:

1. **Send USDT** to the `depositAddress` on the `sourceChain`
2. **Monitor status** using the [Get Bridge Status](/api-reference/bridge/status) endpoint
3. **Check completion** on the destination chain block explorer

<Warning>
  Make sure to send the **exact amount** specified. Partial amounts will not trigger the bridge.
</Warning>

<Info>
  The bridge automatically funds the smart wallet with ETH for gas fees, so you only need to send USDT.
</Info>
